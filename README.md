# Nova Blade Rush

An original space-fantasy, three-lane plasma-blade runner. Dodge raiders and droids while collecting astral crystals.

## Run

Open `index.html` in any modern browser. Tap the left or right side of the game to change lanes; desktop controls are Left/Right arrows or A/D.

The game includes local high scores, daily challenge progress, skins, sound, haptics, sharing, and offline PWA support. No build step or dependencies are required.
